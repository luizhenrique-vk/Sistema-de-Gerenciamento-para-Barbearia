# 💈 Sistema de Gerenciamento para Barbearia

Aplicação web desenvolvida para gerenciar o funcionamento de uma barbearia, permitindo o controle de clientes, serviços e agendamentos de forma organizada e eficiente.

O sistema foi criado com foco em resolver problemas comuns do dia a dia de barbearias, como conflitos de horários, falta de controle de clientes e dificuldade na organização dos atendimentos.

---

## 🚀 Demonstração do Problema

Muitas barbearias ainda utilizam métodos manuais (caderno ou WhatsApp) para controle de horários, o que pode causar:

- ❌ Conflitos de agendamento  
- ❌ Perda de informações de clientes  
- ❌ Falta de organização dos serviços  
- ❌ Dificuldade na gestão do negócio  

Este sistema resolve esses problemas com uma solução digital simples e prática.

---

## 🧩 Funcionalidades

### 👤 Gestão de Clientes
- Cadastro de novos clientes  
- Listagem de clientes cadastrados  
- Organização de informações básicas  

### 📅 Agendamentos
- Criação de agendamentos  
- Controle de horários disponíveis  
- Evita conflitos de horários  

### ✂️ Serviços
- Cadastro de serviços oferecidos  
- Associação de serviços aos atendimentos  

### 🧑‍💼 Profissionais
- Cadastro de barbeiros  
- Organização dos atendimentos por profissional  

### 🔐 Autenticação
- Sistema de login  
- Controle de acesso ao sistema

## 🏗️ Arquitetura

O projeto segue o padrão MVC (Model-View-Controller), separando responsabilidades em:

- **Model** → Representação dos dados  
- **View** → Interface com o usuário  
- **Controller** → Regras de negócio e controle das requisições  

Essa organização facilita manutenção, escalabilidade e entendimento do código.
